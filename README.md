## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository:
	```sh
	git clone <repository-url>
	```
2. Navigate to the project directory:
	```sh
	cd ediyrex
	```
3. Install dependencies:
	```sh
	npm install
	# or
	yarn install
	```

### Running the Project
To start the development server:
```sh
npm run dev
# or
yarn dev
```

### Building for Production
To build the project for production:
```sh
npm run build
# or
yarn build
```

- `src/` - Source code
